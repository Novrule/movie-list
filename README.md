# Movie List

## 프로젝트 소개

```
평점이 높은 영화들의 목록을 보여주고 각각의 영화들에 대한 상세 정보들을 알려줍니다.
```

## 배포 주소

> https://Novrule.github.io/movie-list/

## 프로젝트 주요 기능

> 평점이 높은 영화들의 목록을 보여줍니다.
